Проект на Ruby on Rails "Test Guru"

Описание проекта:

Приложение для создания управления и выполнения тестов:

Роли приложения:
  • Администратор
  • Пользователь

Возможности Администратора:
  1. Разграничение прав доступа
  2. Создание тестов и управление ими
  3. Аутентификация

Возможности пользователя:
  1. Выполнение тестов
  2. Публикация результатов в социальной сети

Онлайн-школа для разработчиков Thinknetica
